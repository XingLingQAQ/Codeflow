# Icon Placeholder
This directory should contain the following icon files for Tauri bundling:

- `32x32.png` - 32x32 PNG icon
- `128x128.png` - 128x128 PNG icon
- `128x128@2x.png` - 256x256 PNG icon (2x scale)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

You can generate these icons from a single high-resolution PNG using:
- Online: https://tauri.app/v1/guides/features/icons
- CLI: `npm run tauri icon path/to/source-icon.png`
