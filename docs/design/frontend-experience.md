## 1. 技术栈

| 层 | 选型 |

保留现有 services/ 与 adapters/ 层资产，仅重写视图层；按 Sessions → Plans → Memory → Plugins 逐视图迁移，每迁一个删一段旧 App.tsx。

## 2. 视觉语言

- 深色优先：低饱和石墨底 + 单一高饱和强调色（电光紫/青）；浮层玻璃拟态（backdrop-blur）；1px 半透明描边替代重阴影
- 字体：Inter/思源黑体 + JetBrains Mono；4/8px 间距栅格；圆角 8/12px 两档
- 动效规范：统一缓动 cubic-bezier(0.32,0.72,0,1)；时长 150/250/400ms 三档；遵守 prefers-reduced-motion
- 微交互：按钮按压 scale(0.97)、聚焦环动画、Toast 弹簧入场、Command Palette 模糊背景淡入、列表 stagger（20ms/项）

## 3. 启动与加载体系（M1）

### 3.1 启动序列（桌面）

```
Tauri 原生启动窗（<100ms 即现，避免白屏）
 → Logo SVG 描边绘制动画 + 渐变流光
 → sidecar 启动阶段进度（后端进程 → 数据库 → 索引加载 → WS 就绪，逐项打勾）
 → 启动窗淡出，App Shell 共享元素接力入场（Logo 飞至导航位）
```

- 启动失败态：进度项变红 + 诊断信息 + 重试/查看日志按钮，杜绝无限转圈
- 网页版等价物：HTML 内联首屏启动页（零 JS 依赖先渲染），bundle 就绪后同样式接力

### 3.2 加载状态分级

| 场景 | 方案 |
|---|---|
| 路由切换 | 顶部进度条（细线流光）+ 路由级骨架屏（与真实布局同构） |
| 面板数据 | 区块骨架 shimmer，禁止全局 spinner |
| 列表分页 | 底部内联 loader + 乐观插入 |
| Agent 流式输出 | 打字光标 + token 渐显 |
| 长任务（索引/导入） | 阶段化进度（步骤名 + 百分比 + 可折叠日志） |
| 空态 | 插画 + 单一主行动按钮 |

## 4. 网页版（M7 收口，M1 起即兼容开发）

- 同一 workbench 构建产物：embed 链路（backend/internal/web）+ 独立部署两种分发
- 能力降级矩阵：无 Tauri API 时 Live Preview 用 iframe 代理；本地文件系统操作禁用，仅远程项目
- 响应式断点：≥1280 全功能 / 768~1280 折叠 Flow Rail、Agent 面板抽屉化 / <768 显示引导页推荐手机伴侣端

## 5. 手机版：远程伴侣端（M7）

### 5.1 定位

不是完整 IDE，是桌面端的**远程控制与监督终端**：移动场景下推进流程、审批、对话。

### 5.2 功能范围

- Flow 监控：项目卡片 → 阶段进度、产物摘要阅读
- **远程审批**：Gate 待审列表（设计稿/Review 报告/守卫豁免），通过/驳回 + 语音批注
- Agent 对话：与 Companion 同一会话续聊（语音输入优先）
- 通知：阶段完成/Gate 等待/辩论结论/构建失败推送
- 远程命令面板：受限白名单命令（触发构建、跑测试、暂停 Agent）
- 只读速览：diff 阅读（移动优化排版）、时间轴回放

### 5.3 连接架构

```
手机端 ──(配对)──> 桌面端（host，运行后端）
  1. 桌面端显示二维码（含一次性配对码 + 公钥指纹）
  2. 手机扫码 → 设备级密钥对交换 → 持久授权（可在桌面端吊销）
  3. 通道：局域网 mDNS 直连优先；跨网走可自建的轻量 relay（仅转发加密帧，零知识）
  4. 协议：复用统一 WS 事件总线，按权限过滤事件子集；E2E 加密
```

### 5.4 技术与安全

- 实现：React Native（与主端共享 packages/shared 类型与部分逻辑）或 Tauri Mobile，开发期可先以 PWA 验证协议
- 安全：审批操作需生物识别确认；远程命令白名单 + 审计日志；会话过期自动断开
